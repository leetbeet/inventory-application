# Inventory Application

A full-stack inventory management system built with **Node.js, Express, PostgreSQL, and EJS**. The application allows users to view, create, update, and delete inventory **items** and **categories** through a simple web interface.

## Features

- View a list of **all categories**
- View **items within a specific category**
- View a list of **all inventory items**
- **Create, edit, and delete categories**
- **Create, edit, and delete items**
- **Server-side validation** using express-validator
- **Password-protected actions** for creating, editing, and deleting data ("Super_Secret_Password123")
- **PostgreSQL relational database**
- **MVC-style architecture** with controllers, models, routes, and views
- Dynamic UI built using **EJS templates and CSS**

## Technologies Used

- **Node.js**
- **Express.js**
- **PostgreSQL**
- **EJS** templating engine
- **express-validator** for input validation
- **dotenv** for environment variables
- **CSS3** for styling
